# Kategori Filtre 

##### Arama inputuna göre filtreleme yapabilirsiniz

##### Seçilen kategori en üstte sıralanır
##### Ara butonuna tıklanınca localstorage de verileri saklar ve sayfaya tekrar gelince seçim işlemleri hatırlanır
